# Comparar Coches

## IMPORTANTE: Búsqueda Online Obligatoria

**NUNCA uses información hardcodeada. SIEMPRE busca información actualizada de ambos coches.**

### Proceso de comparación:

1. **Busca información** de ambos coches en la web
2. **Compara** precio, motor, consumo, equipamiento
3. **Usa una tabla** para mostrar las diferencias
4. **Da un veredicto** basado en el uso del usuario
5. **Pregunta** qué prioriza él
